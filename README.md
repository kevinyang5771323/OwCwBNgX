# 项目概述 python108


## 技术栈

### 后端

- **Python**: 用于编写后端逻辑，处理用户请求、数据库交互及生成动态内容。
- **SQLAlchemy**: 提供高级的对象关系映射（ORM），方便数据库操作。
- **Django 框架**: 利用其路由管理、模板引擎和表单验证等功能，简化开发流程。

### 前端

- **HTML、CSS、JavaScript**: 构建用户界面，可结合React或Vue.js等前端框架。

### 安全性

- 使用Django的表单验证功能，实现前端和后端的数据校验，增强数据安全。

## 功能模块

### 用户

- 用户注册与登录
- 名师信息浏览
- 课程信息查看
- 论坛信息交流
- 试卷列表查阅
- 新闻资讯阅读
- 个人管理后台

### 管理后台

- 首页概览
- 个人中心
- 学生管理
- 名师管理
- 课程名称与信息管理
- 计算机学院与设计学院管理
- 试卷与试题管理
- 论坛管理
- 系统管理

## 系统角色

- 学生：学习课程、参与讨论、提交作业等。
- 教师：发布课程、管理学生、布置作业等。
- 管理员：监控系统、管理用户与内容。

## 运行环境

- **Python版本**: 3.7及以上
- **操作系统**: Windows10/11、MacOS、Linux
- **开发工具**: PyCharm、VSCode
- **数据库**: MySQL 5.7/8.0版本
- **包管理工具**: pip 20.0及以上
- **Web服务器**: Gunicorn 20.0及以上，或uWSGI、Nginx 1.18及以上
